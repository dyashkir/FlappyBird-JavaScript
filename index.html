flappyBird by LearnWD

Fly between pipes, hitting pipe resets score

Code for this can be found at https://github.com/dyashkir/FlappyBird-JavaScript original game can be found at https://github.com/CodeExplainedRepo/FlappyBird-JavaScript there is a video tutorial there as well. I modified the game a bit to make it simpler to inspect with the browser.

You can open you browser Develop menu and click on inspect to see what game is doing. Logic is in flappyBird.js
